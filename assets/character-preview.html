Technical Details

  • ✅ Pure SVG using only <rect> elements
  • ✅ 32×32 pixel grid per character
  • ✅ Solid colors only (no gradients)
  • ✅ Scalable without quality loss
  • ✅ Each character has unique ID
  • ✅ Optimized for Phaser + Electron
  • ✅ Transparent background
  • ✅ 192×32 total spritesheet size

Usage in Phaser

// Load the SVG spritesheet
this.load.svg('characters', 'assets/characters.svg', { width: 192, height: 32 });

// Use individual characters
this.add.image(x, y, 'characters').setFrame('knight');
this.add.image(x, y, 'characters').setFrame('archer');
// etc...
