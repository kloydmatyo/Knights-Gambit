Enemy Details

  • 🟢 Goblin - Weakest enemy, fast and sneaky with a dagger
  • 🟤 Orc - Medium strength, brutish warrior with war axe
  • 💀 Skeleton - Undead fighter with spear, medium difficulty
  • 🟫 Troll - Strongest enemy, massive with club and regeneration

Technical Details

  • ✅ Pure SVG using only <rect> elements
  • ✅ 32×32 pixel grid per enemy
  • ✅ Solid colors only (no gradients)
  • ✅ Scalable without quality loss
  • ✅ Each enemy has unique ID
  • ✅ Matches GameManager enemy stats
  • ✅ 128×32 total spritesheet size

Usage in Phaser

// Load individual enemies
this.load.svg('goblin', 'assets/goblin.svg');
this.load.svg('orc', 'assets/orc.svg');
this.load.svg('skeleton', 'assets/skeleton.svg');
this.load.svg('troll', 'assets/troll.svg');

// Use in combat
const enemy = this.add.image(x, y, 'goblin').setScale(4);
