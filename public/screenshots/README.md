# PWA Screenshots

Add screenshots of your game here to show in app stores and install prompts.

## Required Screenshots:
- `game-1.png` - 540x720px (mobile/narrow)
- `game-2.png` - 1280x720px (desktop/wide)

## Tips:
- Show actual gameplay
- Capture exciting moments (combat, character selection, shop)
- Use high quality images
- Make sure UI is visible and looks good

## How to Capture:
1. Run your game in dev mode
2. Use browser DevTools to set viewport size
3. Take screenshots of key screens
4. Optimize images (use PNG or WebP)
