# PWA Icons

Place your app icons in this directory with the following sizes:

## Required Icons:
- `icon-72x72.png` - 72x72px
- `icon-96x96.png` - 96x96px
- `icon-128x128.png` - 128x128px
- `icon-144x144.png` - 144x144px
- `icon-152x152.png` - 152x152px
- `icon-192x192.png` - 192x192px (Android)
- `icon-384x384.png` - 384x384px
- `icon-512x512.png` - 512x512px (Android)
- `icon-maskable-192x192.png` - 192x192px (with safe zone)
- `icon-maskable-512x512.png` - 512x512px (with safe zone)
- `apple-touch-icon.png` - 180x180px (iOS)
- `favicon.ico` - 32x32px

## Design Guidelines:
- Use your game's dice or character icon
- Background: #0a0602 (dark brown)
- Foreground: #d4a855 (gold)
- For maskable icons, keep important content within 80% safe zone
- Use simple, recognizable design that works at small sizes

## Generate Icons:
You can use tools like:
- https://realfavicongenerator.net/
- https://www.pwabuilder.com/imageGenerator
- https://favicon.io/

Or create them manually with your preferred design tool.
